# Assets

Repositorio de activos visuales y materiales de apoyo del proyecto.

## Organizacion

- `branding/` logos, iconografia y lineamientos visuales
- `diagrams/` exportaciones graficas de arquitectura y flujos
- `screenshots/` capturas para README, demo y evidencia
- `presentations/` presentaciones, pitch deck y apoyo para jurados

## Regla

Todo asset agregado debe tener un uso claro dentro de documentacion, demo o presentacion.
