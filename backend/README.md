# Backend

Modulo de servicios y orquestacion de negocio de EnergiAI.

## Responsabilidad

- exponer contratos REST,
- validar payloads,
- integrar con el servicio ML,
- persistir resultados,
- mantener trazabilidad del flujo.

## Estructura esperada

- `src/main/java/` codigo fuente Spring Boot
- `src/main/resources/` configuracion y recursos
- `src/test/java/` pruebas automatizadas
