# Infra

Modulo de infraestructura y automatizacion operativa del proyecto.

## Organizacion

- `oci/` configuraciones y activos de Oracle Cloud Infrastructure
- `docker/` contenedores y orquestacion local o de demo
- `scripts/` automatizaciones de soporte

## Objetivo

Reducir friccion de despliegue y mantener una ruta de entrega repetible para el MVP.
