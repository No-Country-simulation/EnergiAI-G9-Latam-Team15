# Data

Espacio de trabajo para datos de EnergiAI.

## Organizacion

- `raw/` datos originales no transformados
- `processed/` datos limpiados o transformados
- `samples/` muestras controladas para demo y pruebas

## Reglas

- no versionar datasets sensibles o pesados sin justificacion,
- mantener trazabilidad del origen del dato,
- documentar transformaciones relevantes para el modelo.
