# Dataset Research Report v1

## Objetivo

Identificar fuentes reales de datos energéticos para evitar la construcción completa de datasets sintéticos.

- Comportamientos horarios
- Curvas de carga

https://www.xm.com.co/consumo/historicos-de-demanda

### SIMEM
- Información energética nacional

https://www.simem.co

https://www.inei.gob.pe
